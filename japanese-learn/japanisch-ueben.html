subtitles02 // SYNTAX: A は B です

Das Grundprotokoll: "A ist B". は (wa) ist der Themen-Marker. です (desu) terminiert die Aussage höflich.

infoWICHTIG: Das Hiragana は (ha) wird "wa" ausgesprochen, wenn es als Partikel verwendet wird!

ÜBUNG: DATENPAKET ERSTELLEN

Kompiliere: "Das ist eine Katze."

Assets: これ (kore) = Das, ねこ (neko) = Katze

これ は ねこ です。

quiz03 // INTERROGATIV & NEGATION

Anfrage (か): Suffix か (ka) anfügen.

Negation (ではありません): です (desu) durch ではありません (dewa arimasen) ersetzen.

ÜBUNG: LOGIK INVERTIEREN

Modifiziere zu: "Ist das eine Katze?"

これ は ねこ ですか。

Modifiziere zu: "Das ist keine Katze."

これ は ねこ ではありません。

link04 // POSSESSIV-PARTIKEL: の (no)

Besitz-Zuweisung: [Besitzer] の [Besitz].

ÜBUNG: ASSET-ZUWEISUNG

Kompiliere: "meine Katze"

Assets: わたし (watashi) = ich/mein

わたし の ねこ

Kompiliere den Satz: "Das ist meine Katze."

これ は わたし の ねこ です。

directions_run05 // VERBEN & PARTIKEL を (o)

Jetzt kommen Handlungen ins Spiel. Verben stehen immer am Ende des Satzes und enden in der höflichen Form auf -masu (-ます).

Verb Hiragana Bedeutung
tabemasu たべます essen
nomimasu のみます trinken
mimasu みます sehen / schauen
kaimasu かいます kaufen

Der Partikel を (o) markiert das direkte Objekt – also das, worauf sich die Handlung bezieht.

infoAussprache: Das Zeichen を wird wie お ("o") ausgesprochen.

ÜBUNG: AKTIONEN AUSFÜHREN

Übersetze: "Ich schaue Anime."

わたし は アニメ を みます。

Übersetze: "Ich trinke Wasser."

Assets: みず (mizu) = Wasser

わたし は みず を のみます。

Übersetze: "Ich kaufe ein Buch."

Assets: ほん (hon) = Buch

わたし は ほん を かいます。

palette06 // ADJEKTIVE: DINGE BESCHREIBEN

Es gibt zwei Arten von Adjektiven: i-Adjektive und na-Adjektive. Ihre grammatikalische Anwendung ist unterschiedlich.

1. i-Adjektive (い-Adjektive)

Enden in ihrer Grundform auf い (i).

  • あたらしい (atarashii) - neu
  • おもしろい (omoshiroi) - interessant, lustig
  • おいしい (oishii) - lecker
  • たかい (takai) - teuer / hoch

Verwendung vor Nomen: おもしろい ほん (ein interessantes Buch).

Verwendung am Satzende: この ほん は おもしろい です。 (Dieses Buch ist interessant).

2. na-Adjektive (な-Adjektive)

Benötigen ein な (na), wenn sie vor einem Nomen stehen.

  • しずか (shizuka) - ruhig, leise
  • げんき (genki) - gesund, munter
  • きれい (kirei) - schön, sauber (Ausnahme!)

Verwendung vor Nomen: しずかな へや (ein ruhiges Zimmer).

Verwendung am Satzende (ohne な): この へや は しずか です。 (Dieses Zimmer ist ruhig).

ÜBUNG: EIGENSCHAFTEN ZUWEISEN

Wie sagst du "ein neues Buch"?

あたらしい ほん

Wie sagst du "ein schönes Bild"?

Assets: きれい (kirei), え (e) = Bild

きれいな え

Übersetze: "Dieses Essen ist lecker."

Assets: この (kono), たべもの (tabemono) = Essen, おいしい (oishii)

この たべもの は おいしい です。
